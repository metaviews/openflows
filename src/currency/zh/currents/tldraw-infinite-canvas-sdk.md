**Signal 信号**
无限画布 SDK 带内置 AI 启动套件 (starter kits) · opensourceprojects · 2026-04-26

tldraw SDK 提供全功能画布组件，专为白板风格的无限画布设计，包含启动套件，用于将 AI 绘图与编辑能力直接集成至用户应用中。

**Context 语境**
随着 AI 界面超越基于文本的聊天演进，空间化与协作式工作区正成为主要的交互模式。无限画布工具允许非结构化的创意生成与绘图，这与智能体 (agent) 推理及多步规划的非线性本质相契合。

**Relevance 关联**
本条目记录了智能体界面层从线性提示到空间语境的转变。通过提供内置 AI 启动套件，该 SDK 降低了将生成式能力集成到自定义协作环境中的摩擦，支持本地优先 (local-first) 或自托管 (self-hosted) 智能体工作区的架构模式。

**Current State 当前状态**
该项目为开源 (open source)，拥有公开的 GitHub 仓库。SDK 包含全功能画布组件，以及针对 AI 绘图与编辑的特定集成套件，表明其在开发者工具与智能体前端方面的即时适用性。

**译注**
1. **Agent (智能体)**：此处译为“智能体”而非“代理”，以强调其作为修行者 (practitioner) 在生态中的能动性，而非单纯的工具。
2. **Inference (推理)**：与“理” (lǐ) 共享字符，暗示推理过程需遵循事物的自然纹理 (grain)。
3. **Starter kits (启动套件)**：保留英文术语，指代开箱即用的集成模块，中文“套件”未能完全传达其“快速启动”的意味。
4. **Current (流)**：本条目类型为 `current`，在 Openflows 语境下对应“流” (liú)，指代生态中流动的、正在发生的信号，而非静态的“回路” (circuit)。
